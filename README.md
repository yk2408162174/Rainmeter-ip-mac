# Rainmeter 网络监控皮肤 (Network Monitor)

这是一个基于 Rainmeter 的轻量级网络监控皮肤，支持实时显示 IP、MAC 地址及网卡状态。

## 项目结构
本项目包含两个迭代版本：

* **`/Version_1`**: 基础版。提供手动刷新机制，代码简单易读，适合初学者研究。
* **`/Version_2`** (推荐): 进阶版。引入了 PowerShell 事件监听机制，实现**全自动网卡状态监测**。无需手动操作，检测到网络切换（Wi-Fi/以太网）即自动更新显示。

## 功能特点
- **自动监测**：基于 Windows WMI 事件驱动，零延迟捕捉网络状态变动。
- **编码优化**：统一使用 UTF-16 LE 编码，彻底解决中文乱码问题。
- **配置集中**：所有设置项均集成在 `Variables.inc` 中，方便维护。

## 快速上手
1. 下载整个仓库到本地。
2. 将对应版本的文件夹放入 Rainmeter 的 `Skins` 目录下。
3. 右键点击 Rainmeter 托盘图标并刷新。
4. 进阶版在首次加载后，即进入后台自动化监控模式。
